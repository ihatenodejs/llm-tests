### How to use

1. **Copy** the entire code block into a new file called `index.html`.
2. **Open** that file in your browser – you’ll see the full site.
3. **Deploy**  
   * GitHub Pages: push to a repository named `yourusername.github.io`.  
   * Netlify/Vercel: drag the file into the “drag & drop” area.  
   * Your own server: upload `index.html` to the web root.

### Customization tips

| What you might want to change | Where |
|------------------------------|-------|
| Logo or shop name            | `<h1>` inside `<header>` |
| Color scheme                 | Edit the CSS variables in `<style>` (e.g., `#0066cc` for links & buttons) |
| Images                        | Replace the Unsplash URLs with your own photos or local images. |
| Contact form backend         | In the `<script>` section, replace `console.log` and `alert` with an AJAX call to your server or a service like Formspree. |
| Additional pages             | Add new `<section id="…">` blocks and corresponding navigation links. |
