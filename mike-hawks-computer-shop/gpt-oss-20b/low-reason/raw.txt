## 3. `app.js` (optional, basic form validation)

```js
document.getElementById('contact-form').addEventListener('submit', function (e) {
  e.preventDefault(); // Prevent actual submission for demo

  const name = this.name.value.trim();
  const email = this.email.value.trim();
  const message = this.message.value.trim();

  if (!name || !email || !message) {
    alert('Please fill in all fields.');
    return;
  }

  // Simulate successful send
  alert(`Thank you, ${name}! Your message has been sent.`);
  this.reset();
});
```

### How to Deploy

1. Create a folder (e.g., `mikeshop`).
2. Add the three files above (`index.html`, `style.css`, `app.js`).  
   *(You can also add a small SVG for the favicon.)*
3. Open `index.html` in any browser to view locally.
4. Upload the folder to your web host or a static site service (Netlify, GitHub Pages, Vercel).

**Result:** A clean, functional website that clearly communicates Mike Hawk’s unique policy – no Windows repairs – while offering a professional look and feel. Feel free to tweak colors, fonts or content as you wish!
